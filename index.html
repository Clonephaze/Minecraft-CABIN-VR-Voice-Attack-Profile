CABIN VR Command Sheet by Clonephaze

Understanding the Command Sheet

If you're new to dynamic command structures, don't worry—it's simpler than it looks. Let's break it down using the backpack command as an example. The format "[Open;Close] [Bag;Backpack]" means you can say "Open bag," "Open backpack," "Close bag," or "Close backpack." Brackets indicate interchangeable words, and if a pair of brackets has a semicolon at the end (e.g., "[item;]"), you can leave it out entirely.

If voice commands are disabled, you can still trigger any command by first saying "CABIN" followed by the command.

Required Keybindings

Some commands rely on specific keybindings. You can either use the recommended bindings below in minecraft or customize them in VoiceAttack:

  • Unbind Options GUI (chunk pregenerator)
  • Unbind COFH Cycle Mode Down
  • Bind Quick Torch → '
  • Bind Show Keyboard → HOME
  • Bind Open Wireless Terminal → Page Up
  • Bind Balance Grid → SHIFT + /
  • Bind Clear Grid → SHIFT + *
  • Bind Quick Add Waypoint → End
  • Bind Waypoint Manager → Delete
  • Bind Quests → `
  • Bind Open Satchel → Page Down
  • Bind Toggle Movement Mode → 0 (Zero)

Commands

[Go; Go to] Home [Location;]

Sends the player to their set home location.

  • Begin Boolean Compare : [ttsEnabled] Equals True, Say, 'Sending you home' , Else, Play sound, 'internal:Cleek' , End Condition, Press / ? key and hold for 0.01 seconds and release, Pause 0.1 seconds, Quick Input, 'home', Press Enter key and hold for 0.01 seconds and release

Go Back

Goes Back to previous location before using the /home command.

  • Begin Boolean Compare : [ttsEnabled] Equals True, Say, 'Sending you back' , Else, Play sound, 'internal:Cleek' , End Condition, Press / ? key and hold for 0.01 seconds and release, Pause 0.1 seconds, Quick Input, 'back', Press Enter key and hold for 0.1 seconds and release

Set Home [Location;]

Sets the player's home location to their current position.

  • Begin Boolean Compare : [ttsEnabled] Equals True, Say, 'Set the users Home Location' , Else, Play sound, 'internal:Cleek' , End Condition, Press / ? key and hold for 0.01 seconds and release, Pause 0.1 seconds, Quick Input, 'set home', Press Enter key and hold for 0.01 seconds and release

Set Gamemode to [Survival;Creative;Adventure;Spectator]

Switch to Creative, Survival, Adventure, or Spectator mode.

  • Begin Boolean Compare : [ttsEnabled] Equals True, Say, 'Switching Game Mode' , Else, Play sound, 'internal:Cleek' , End Condition, Set text [mode_selection] to '{CMDSEGMENT:1}', Press / ? key and hold for 0.01 seconds and release, Pause 0.1 seconds, Quick Input, 'gamemode ', Quick Input, '{TXT:mode_selection}', Press Enter key and hold for 0.1 seconds and release

Take a screenshot

Takes a screenshot

  • Press F2 key and hold for 0.1 seconds and release

Hotbar and Overworld

Hotbar Slot [1..9]

Switches the active hotbar slot to the number specified.

  • Set text [SlotNumber] to '{CMDSEGMENT:1}', Quick Input, '{TXT:slotNumber}', Play sound, 'internal:Cleek'

Drop [this;these;] [item;items;stack]

Drops an item or stack of items based on user request.

  • Set text [dropItemCount] to '{CMDSEGMENT:2}', Begin Text Compare : [{TXT:dropItemCount}] Equals 'stack', Press Left Ctrl+Q keys and hold for 0.1 seconds and release, Begin Boolean Compare : [ttsEnabled] Equals True, Say, 'Dropping Stack' , Else, Play sound, 'internal:Cleek' , End Condition, Else, Press Q key and hold for 0.1 seconds and release, Begin Boolean Compare : [ttsEnabled] Equals True, Say, 'Dropping Item' , Else, Play sound, 'internal:Cleek' , End Condition, End Condition

[Swap;Switch] Offhand [item;]

Swaps whatever item is highlighted in your hotbar with your offhand slot.

  • Press F key and hold for 0.1 seconds and release, Play sound, 'internal:Cleek'

[Hold;Toggle;Stop Holding] Control

Many times mods want you to hold control, this allows you to toggle it in VR. Will auto turn off after ten seconds.

  • Press F key and hold for 0.1 seconds and release, Play sound, 'internal:Cleek'

[Open;Toggle;Close] Quest* [book;]

Opens the quest book. For this one, "Open Quests" also works.

View actions
  • Press ` ~ key and hold for 0.1 seconds and release

MOD - Create

[Open;Toggle;Close] [nearby;] Toolboxes

Many Create machines want you to hold right click to change settings, but that doesn't work well in VR. This works though.

  • Set text [createInteract] to '{CMDSEGMENT:0}', Begin Text Compare : [{TXT:createInteract}] Equals 'Hold', Hold right mouse button down, Else, Release right mouse button, End Condition

Initialize CABIN Profile

Runs on profile startup.

  • ::Sets the Text to Speech variable to false on first profile startup, Begin Boolean Compare : [ttsEnabled] Has Not Been Set, Set Boolean [ttsEnabled] to False (save value to profile), End Condition

[Start;Begin] [with;] [Typing;Dictation]

Begins listening for dictation. Useful for typing in search bars quickly, but try simple phrases unless your recognition is well trained.

Toggle [tts;Text to Speech] [responses;]

Toggles TTS responses for any commands that use them.

  • Begin Boolean Compare : [ttsEnabled] Equals False, Set Boolean [ttsEnabled] to True (save value to profile), Say, 'Text to Speech responses enabled', Else, Set Boolean [ttsEnabled] to False (save value to profile), Play sound, 'internal:Cleek', End Condition
